Helio reads all configuration from the environment. The authoritative,
always-current list is [`.env.example`](https://github.com/achref-soua/helio/blob/main/.env.example)

  Never commit real secrets â€” inject them through your environment or a secrets manager.

### AI security

Keys are read as secrets and never logged. Prompts are refused over plaintext
HTTP to a non-local endpoint unless you explicitly opt out. Copilot retrieval
runs over the RLS-bound application role, never the admin connection â€” so the
model can never reach another tenant's data. For full data sovereignty, point
`INTEL_LLM_PROVIDER` at `ollama` or `local`.
