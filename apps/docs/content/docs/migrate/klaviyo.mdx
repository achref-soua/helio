title: Migrate from Klaviyo
description: Pull profiles straight from Klaviyo with a private API key — marketing consent honored.
---

## One-click: connect Klaviyo

1. In Klaviyo, create a **private API key** (Settings → API keys) with
   read access to profiles.
2. In Helio, **Settings → Provider credentials → Add** → kind **Klaviyo**,
   paste the key (vault-sealed, shown once).
3. **Contacts → Import CSV → Or pull straight from a platform** — pick the
   connection and **Pull contacts**.

Helio pages through your profiles with email, names, organization (which
becomes a Helio **company**), and email marketing consent — a profile
whose consent is `UNSUBSCRIBED` lands suppressed and is never mailed.

## CSV fallback

Export profiles from any list or segment and drop the file on the import
wizard — Klaviyo exports are detected automatically, and the mapping step
shows exactly where each column lands before anything is written.
