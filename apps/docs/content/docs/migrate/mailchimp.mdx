title: Migrate from Mailchimp
description: Pull every audience straight from Mailchimp with an API key — statuses mapped, unsubscribes honored.
---

## One-click: connect Mailchimp

1. In Mailchimp, create an **API key** (Account → Extras → API keys). Keys
   end in your datacenter, e.g. `…-us21`.
2. In Helio, **Settings → Provider credentials → Add** → kind
   **Mailchimp**, paste the key (vault-sealed, shown once).
3. **Contacts → Import CSV → Or pull straight from a platform** — pick the
   connection and **Pull contacts**.

Helio walks **every audience** and imports members with their names and
status. The status mapping is strict where it matters:

| Mailchimp status | In Helio                        |
| ---------------- | ------------------------------- |
| `subscribed`     | Active                          |
| `pending`        | Active                          |
| `unsubscribed`   | **Unsubscribed** (never mailed) |
| `cleaned`        | **Unsubscribed** (never mailed) |

## CSV fallback

Audience → Export audience. The wizard detects Mailchimp exports and maps
`Email Address`, names, and the status column automatically — adjust
anything in the mapping step.
